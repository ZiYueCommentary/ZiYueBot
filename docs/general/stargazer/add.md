# 添加星标 {#add-stargazer}

**添加星标 (Add stargazer)** 可以对云瓶进行星标操作，将其加入用户的星标列表。单个云瓶只能星标一次。

除调用此命令外，还可以对云瓶消息回复 👍 或 <Badge type="warning" text="仅 Discord" /> ⭐ 表情来添加星标。

## 用法 {#usage}

```
/添加星标 [id]
```

## 参数 {#params}

* `id` 是云瓶编号。

## 输出 {#output}

* 正常调用时：

    ```
    {用户} 已星标 {瓶子编号} 号云瓶！对云瓶消息回应“点赞”图标即可星标~
    ```

* 重复星标时：

    ```
    {用户}：您已星标过 {瓶子编号} 号云瓶
    ```


## 频率限制 {#rate-limit}

每次有效调用间隔不小于 1 分钟。
