# 关于 {#about}

**关于 (About)** 是一个 [鸿蒙命令](/harmony/)，可以获得子悦机器的相关信息。

## 用法 {#usage}

```
/about
```

## 参数 {#params}

无

## 输出 {#output}

```
子悦机器 (ZiYue Bot) - {版本号}
子悦机器是一个由 子悦解说 开发的，用 C# 编写的 QQ 和 Discord 机器人。
使用教程：https://docs.ziyuebot.cn/usage.html
开源仓库：https://github.com/ZiYueCommentary/ZiYueBot
在线文档：https://docs.ziyuebot.cn/
```

## 频率限制 {#rate-limit}

无
