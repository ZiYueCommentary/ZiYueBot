# :heart: 赞助者 {#sponsors}

**赞助者 (Sponsor)** 是在 [子悦爱发电](https://afdian.com/a/ziyuecommentary2020) 赞助“子悦机器”方案的爱发电用户。

每次赞助的过期时间为 365 天之后，无视赞助时长。赞助生效时，爱发电会将用户填写的聊天平台账号加入赞助者列表。

子悦机器的赞助者可以使用 [绘画](general/draw) 命令。
