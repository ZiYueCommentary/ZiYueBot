# 命令别名 {#alias}

**命令别名 (Alias)** 是一种提升用户使用体验的设计，用于在命令名被混淆时（例如：[开始俄罗斯轮盘](../harmony/revolver/start)、开启俄罗斯轮盘），提示用户输入正确的命令名。为了纠正用户的使用习惯，引导用户使用唯一正确的命令名，该机制不会将调用请求传给正确的命令。

该机制只在 QQ 存在，因为 Discord 中有明确的命令选择界面，可以引导用户使用正确名称。

命令别名配置文件位于 `resources/alias.json` 中，数据结构如下：

```json
{
  "命令1": [
    "别名1",
    "别名2"
  ],
  "命令2": [
    "别名1"
  ]
}
```

当子悦机器找到别名时：

```
命令未找到，你是否在找 /{命令名}？
```
